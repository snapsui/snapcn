### Props

| Prop      | Type                    | Required | Default | Description                                      |
| --------- | ----------------------- | -------- | ------- | ------------------------------------------------ |
| value     | string                  | Yes      | -       | The current value of the input                   |
| onChange  | (value: string) => void | No       | -       | Callback function called when the value changes  |
| className | string                  | No       | -       | Additional CSS classes to apply to the component |
