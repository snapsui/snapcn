## Props

| Prop          | Type                         | Default | Description                                                                                         |
| ------------- | ---------------------------- | ------- | --------------------------------------------------------------------------------------------------- |
| value         | string                       | ""      | The current value of the textarea.                                                                  |
| onChange      | (value: string) => void      | -       | Callback function invoked when the textarea value changes.                                          |
| leftIcon      | React.ReactNode              | -       | An icon to be displayed on the left side of the textarea.                                           |
| maxCharLength | number                       | -       | The maximum number of characters allowed in the textarea. Displays a character count when provided. |
| className     | string                       | ""      | Additional classes to style the textarea.                                                           |
| placeholder   | string                       | ""      | Placeholder text to display when the textarea is empty.                                             |
| disabled      | boolean                      | false   | If true, the textarea will be disabled and cannot be interacted with.                               |
| rows          | number                       | 3       | The number of visible text lines for the textarea.                                                  |
| autoFocus     | boolean                      | false   | If true, the textarea will automatically get focus when the page loads.                             |
| name          | string                       | -       | The name attribute of the textarea.                                                                 |
| ref           | React.Ref                    | -       | Reference to the textarea element, for handling DOM-related functions.                              |
| ...props      | React.TextareaHTMLAttributes | -       | Inherits all standard HTML textarea attributes.                                                     |
