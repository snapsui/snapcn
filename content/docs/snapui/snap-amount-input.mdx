### Props

| Prop             | Type                             | Default | Description                                                     |
| ---------------- | -------------------------------- | ------- | --------------------------------------------------------------- |
| codeValue        | string                           | -       | The currency code to be displayed and selected in the dropdown. |
| onCurrencyChange | (value: string) => void          | -       | Callback function triggered when a currency is selected.        |
| value            | number                           | string  | -                                                               |
| onChange         | event: React.ChangeEvent => void | -       | Callback function triggered when the input value changes.       |
| placeholder      | string                           | -       | Placeholder text for the input field.                           |
| disabled         | boolean                          | false   | If true, the input will be disabled.                            |
| className        | string                           | -       | Additional CSS classes to apply to the component.               |
