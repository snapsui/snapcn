### Props

| Prop        | Type                        | Required | Default                 | Description                                              |
| ----------- | --------------------------- | -------- | ----------------------- | -------------------------------------------------------- |
| value       | string                      | Yes      | ""                      | The current value of the phone number input.             |
| onChange    | (value: E164Number) => void | Yes      | undefined               | Callback function invoked when the phone number changes. |
| placeholder | string                      | No       | "Enter phone number..." | Placeholder text for the input field.                    |
| className   | string                      | No       | ""                      | Additional CSS classes for styling customization.        |
| autoFocus   | boolean                     | No       | false                   | Sets focus on the input automatically when true.         |
