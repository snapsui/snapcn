## Props

| Prop        | Type            | Required | Default | Description                                           |
| ----------- | --------------- | -------- | ------- | ----------------------------------------------------- |
| className   | string          | No       |         | Additional CSS classes to apply to the input wrapper. |
| type        | string          | No       | "text"  | The type of the input element.                        |
| leftIcon    | React.ReactNode | No       |         | Icon to display on the left side of the input.        |
| rightIcon   | React.ReactNode | No       |         | Icon to display on the right side of the input.       |
| hint        | string          | No       |         | Hint text to display below the input.                 |
| disabled    | boolean         | No       | false   | Whether the input is disabled.                        |
| placeholder | string          | No       |         | Placeholder text for the input.                       |
| prefix      | string          | No       |         | Text to display as a prefix before the input.         |
| ref         | React.Ref       | No       |         | Ref to the underlying input element.                  |
