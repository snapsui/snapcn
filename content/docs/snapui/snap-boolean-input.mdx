## Props

| Prop      | Type                        | Required | Default | Description                                              |
| --------- | --------------------------- | -------- | ------- | -------------------------------------------------------- |
| value     | boolean                     | Yes      | -       | The current boolean value of the input                   |
| onToggle  | (newValue: boolean) => void | No       | -       | Callback function called when the value is toggled       |
| readonly  | boolean                     | No       | `false` | If true, the input becomes read-only and non-interactive |
| className | string                      | No       | -       | Additional CSS classes to apply to the component         |
| disabled  | boolean                     | No       | `false` | If true, the input is disabled and non-interactive       |

The component also accepts all standard HTML button attributes.
