## Props

| Prop      | Type                    | Required | Default | Description                                           |
| --------- | ----------------------- | -------- | ------- | ----------------------------------------------------- |
| className | string                  | No       |         | Additional CSS classes to apply to the input wrapper. |
| value     | number                  | Yes      |         | The value of the rating input.                        |
| onChange  | (value: number) => void | No       |         | Callback function to call when the value changes.     |
| readonly  | boolean                 | No       | false   | Whether the input is readonly.                        |
| ref       | React.Ref               | No       |         | Ref to the underlying input element.                  |
