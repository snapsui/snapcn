## Props

| Prop      | Type                   | Description                                                                                       |
| --------- | ---------------------- | ------------------------------------------------------------------------------------------------- |
| leftIcon  | React.ReactNode        | An icon to be displayed on the left side of the input field.                                      |
| rightIcon | React.ReactNode        | An icon to be displayed on the right side of the input field.                                     |
| hint      | string                 | A small hint or additional information to be displayed below the input field. Visible on hover.   |
| className | string                 | Additional classes to style the input field.                                                      |
| type      | React.HTMLInputElement | Specifies the input field's type (e.g., text, password, email).                                   |
| ref       | React.Ref              | Reference to the input element, for handling DOM-related functions.                               |
| props     | React.HTMLInputElement | TInherits all standard HTML input attributes such as placeholder, value, onChange, disabled, etc. |
