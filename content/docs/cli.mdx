## Getting Started

Begin by using the `init` command to set up the necessary dependencies for your new project.

The `init` command will install essential packages like `framer-motion`, configure `tailwind.config.js`, and add the `cn` utility as well as CSS variables for your project.

```bash
npx snapcn init
```

### For Projects Already Using ShadCN-UI

No need to worry if you're already using `shadcn-ui` in your project! SnapUI is fully compatible.

```bash
npx shadcn-ui init
```

Just add the following lines to your `components.json` file:

```diff
{
  "$schema": "https://ui.shadcn.com/schema.json",
  "style": "default",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "app/globals.css",
    "baseColor": "slate",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
+   "ui": "@/components/ui",
+   "magicui": "@/components/magicui"
  }
}
```

## Adding Components

To add a component to your project, use the `add` command. This will automatically install the component and any required dependencies.

```bash
npx snapcn add [component]
```

### Example Usage

```bash
npx snapcn add snap-input
```

If you'd like to install all available components, use the `--all` flag:

```bash
npx snapcn add --all
```

For installing a specific example or demo that you saw on the website, use the `--example` flag:

```bash
npx snapcn add --example
```

Running the `add` command without any arguments will show you a list of available components:

```bash
npx snapcn add
```

## ShadCN-UI Integration

The CLI can also be used to install and manage `shadcn-ui` components:

```bash
npx snapcn add --shadcn button
```

```bash
npx snapcn add --shadcn --all
```
